About Rising Waters

An AI-powered flood prediction system using Machine Learning, Flask and XGBoost.

🌊 Project Overview

Floods are among the most devastating natural disasters, causing widespread damage to life, infrastructure and the environment.

This project uses Machine Learning to estimate flood risk based on five important environmental parameters and provides an instant prediction through a Flask web application.

The prediction model is trained using the XGBoost algorithm after comparing multiple classification models.

🎯 Objectives

  • Predict flood risk accurately.
  • Support early warning systems.
  • Help disaster management authorities.
  • Reduce response time during emergencies.
  • Demonstrate practical Machine Learning deployment.
